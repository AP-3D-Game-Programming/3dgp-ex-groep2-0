In ons spel Dead End Drive gebruiken wij 1 grote core mechanic.
Je komt een huis binnen, doet daar iets, dan rij je verder om vervolgens het zelfde huis binnen te moeten. Maar er is 1 ding veranderd, waar je dus iets nieuws moet doen.

Om deze mechanic te testen met een paper prototype zullen we een soort zoek de verschillen uitproberen.
We gebruiken als eerst een foto van een plattegrond van een huis. Deze laten we even instuderen door de groep. Vervolgens geven wij een nieuwe foto van dit huis, alleen is er iets veranderd (keuken omgedraaid, planten verwisseld, fiets weg, etc...). De groep moet dus zeggen wat er is veranderd.

Door elke keer een nieuwe foto te laten waarvan de verandering kleiner en kleiner is, zullen we testen hoe klein we een verandering kunnen maken voor het frustrerend wordt om deze te vinden.

In de map "foto's" kan je alle foto's vinden die zullen gebruikt worden. Basis.png is het hoofdplattegrond, bij de andere is er iets veranderd (in de naam staat wat).
